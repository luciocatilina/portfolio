My projects

Surnames in Argentina

It is a Django web app where the user inputs a last name, and an interactive map is displayed showing the frequency of that surname in Argentina.

The app utilizes MongoDB integrated with pymongo for data retrieval and storage, instead of using Django ORM.

Principal tools: Python, Django, JavaScript, JS Ajax, pymongo

Forum

It is a Django web app that serves as a forum where users can create posts and comment on each other's posts. Additionally, users have the ability to add posts to their "favorites" list and view them later. The is a crowdfunding system with Mercado Pago The app is currently in development and includes pending tasks such as integrating CKEditor into the post creation template.

Static files are served through Nginx, while the rest of the application is handled by Apache. The database being used is MariaDB.

Principal tools: Python, Django, JavaScript, JS Ajax, Scss, Bootstrap

JavaScript Cart
